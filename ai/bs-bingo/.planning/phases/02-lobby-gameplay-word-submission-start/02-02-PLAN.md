**onMessage switch — add 4 new cases after `case "ping"` (after line 108):**

`case "submitWord"`: Extract `text` from result.output. Trim it: `const normalized = text.trim()`. Check duplicate: `const exists = [...this.#words.values()].some(w => w.text.toLowerCase() === normalized.toLowerCase())`. If exists, `conn.send(JSON.stringify({ type: "error", code: "duplicate_word", message: \`"${normalized}" is already in the pool\` }))` and return. Otherwise, generate `const wordId = nanoid()`, get `const connState = conn.state as { playerId?: string } | null`, create entry `{ wordId, text: normalized, submittedBy: connState?.playerId ?? "unknown" }`, set `this.#words.set(wordId, entry)`, broadcast `{ type: "wordAdded", word: entry }`. CRITICAL: no `await` between dedupe check and `#words.set()` (Pitfall 4).

`case "removeWord"`: Extract `wordId`. Get entry from `this.#words.get(wordId)`. If not found, return (idempotent). Get `connState = conn.state as { playerId?: string } | null`. If `entry.submittedBy !== connState?.playerId`, send `{ type: "error", code: "not_owner" }` and return. Otherwise, `this.#words.delete(wordId)` and broadcast `{ type: "wordRemoved", wordId }`.

`case "loadStarterPack"`: Get connState. If `connState?.playerId !== this.#hostId`, return silently (host-only guard). Extract `pack`. If `this.#usedPacks.has(pack)`, return silently (once-per-session per D-09). Add to `this.#usedPacks.add(pack)`. Get `packWords = STARTER_PACKS[pack as keyof typeof STARTER_PACKS]`. Loop over packWords: for each text, check duplicate (same `.toLowerCase()` comparison), if not duplicate, create entry with `submittedBy: connState.playerId!` (NOT "pack" — Pitfall 3 from RESEARCH.md — host must be able to delete pack words per D-02), set in `#words`, broadcast `wordAdded`. 

`case "startGame"`: Get connState. If `connState?.playerId !== this.#hostId`, return silently. If `this.#words.size < 5`, send `{ type: "error", code: "not_enough_words" }` and return. Set `this.#phase = "playing"`. Broadcast `{ type: "roomState", state: this.#snapshot() }`.
